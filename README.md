## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Navigate to the project folder:

```bash

```

Install dependencies:


Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
